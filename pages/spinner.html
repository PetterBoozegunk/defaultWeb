spinners.js uses "spin.js"

  2. 
    A spinner will get the same color as the parent element (css) color.
    To get the right color in ie8 and below the color has to be a hex value (ex: '#ff0000/#f00' not 'red')
    <div class="spinnerTest2"> <!-- --> </div> $(".spinnerTest2").trigger("spinner:get");
  3. <div class="spinnerTest3"> <!-- --> </div> 
    
 The spinnerObj will overwrite the spinner defaults
var spinnerObj = {
    color: "#90ee90"  lightgreen
};

$(".spinnerTest3").trigger("spinner:get", spinnerObj);
